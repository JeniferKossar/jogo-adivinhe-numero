Sobre o NutriBem

O NutriBem é um projeto de site com foco no agendamento de consultas nutricionais, desenvolvido como parte da Atividade Somativa de Fundamentos de Programação Web. Nosso objetivo é facilitar o acesso a profissionais de nutrição, permitindo que usuários marquem consultas de forma simples e intuitiva, seja online ou presencialmente.

Este site demonstra a aplicação de tecnologias front-end fundamentais, como HTML para estruturação de conteúdo, CSS para estilização visual e JavaScript para interatividade e validação de dados, sem o uso de frameworks ou bibliotecas externas.

Funcionalidades Simuladas:

  • Agendamento de Consultas: Através de um formulário com validação de campos.
  • Visualização de Dados: Os dados do formulário são exibidos em uma página de confirmação (formAction.html).
  • Navegação Consistente: Menu presente em todas as páginas para fácil acesso.

Nota: As funcionalidades de login com redes sociais, painel do paciente com histórico de dietas e medidas são conceituais neste protótipo, pois exigiriam integração com back-end e bancos de dados, que estão fora do escopo desta atividade.

Utilizado para a marcação semântica e estruturação de todas as páginas do site.

Responsável por toda a apresentação visual, incluindo cores, fontes, layout e responsividade.

Empregado para adicionar interatividade, como a validação de todos os campos do formulário.
